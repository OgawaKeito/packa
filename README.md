# Packa（パッカ） - 旅の準備を爆速にする共有荷造りリスト

**Packa（パッカ）** は、旅行の宿泊数や目的地（国内・海外）に合わせて持ち物リストを自動生成し、URLひとつで仲間と共有・編集できるWebアプリケーションです。

アプリのインストール不要、ブラウザだけで完結するシンプルな設計で、グループ旅行や家族旅行の準備をスムーズにします。

---

## 主な機能

- **リスト自動生成**
  「国内 / 海外」と「宿泊数」を選択するだけで、旅行スタイルに最適な持ち物リストを自動で提案します。

- **自由な編集 & 0から作成**
  アイテムの追加・削除は自由自在。「自分で0から作成する」モードも搭載。

- **ドラッグ&ドロップ並べ替え**
  アイテムの順番を直感的に入れ替え可能。スマホのタッチ操作にも対応。並び順はデータベースに保存され、共有相手にも同じ順番で表示されます。

- **カンタン共有**
  発行されたURLをLINEなどで送るだけ。ログイン不要で誰でも閲覧・編集が可能。「URLをコピー」「URLを開く」ボタンでスムーズに共有できます。

- **レスポンシブデザイン**
  PC・タブレット・スマートフォン（iOS / Android）のどの端末でも見やすく操作しやすいUIを提供します。

- **リアルタイム保存**
  チェックボックスの状態やリストの変更は即座にデータベースへ反映されます。

## 使用技術

### フロントエンド

| 技術 | 用途 |
|---|---|
| HTML5 / CSS3 | モバイルファーストなレスポンシブデザイン |
| Vue.js 3（CDN版） | リアクティブなUI操作（ビルド不要） |
| SortableJS | 滑らかなドラッグ&ドロップ機能 |

### バックエンド

| 技術 | 用途 |
|---|---|
| PHP 7.4+ | シンプルなREST APIエンドポイント |
| MySQL / MariaDB | リストデータ・マスタデータの管理 |

## セットアップ

### 1. ファイルの配置

リポジトリをクローンし、Webサーバー（Apache / Nginx）のドキュメントルート配下に配置します。

```bash
git clone https://github.com/your-username/packa.git
```

### 2. データベースの準備

MySQL / MariaDB にデータベースとテーブルを作成します。

### 3. db_config.php の作成

`db_config.php` は `.gitignore` により管理対象外となっています。以下の内容を参考に、手動で作成してください。

```php
<?php
define('DB_HOST', 'localhost');
define('DB_NAME', 'your_database_name');
define('DB_USER', 'your_database_user');
define('DB_PASS', 'your_database_password');
define('DB_CHARSET', 'utf8mb4');
```

### 4. 動作確認

ブラウザで `index.html` にアクセスし、リストの生成・保存・共有が正常に動作することを確認します。

---

## 注意事項

- `db_config.php` にはデータベースの認証情報が含まれるため、**絶対にGitにコミットしないでください**。`.gitignore` により追跡対象から除外されています。
- 本アプリはログイン機能を持たないため、発行されたURLを知っている人は誰でもリストを閲覧・編集できます。機密性の高い情報の管理には使用しないでください。
