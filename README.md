# 📦 Packa - 旅の準備を爆速にする共有荷造りリスト

**Packa（パッカ）** は、旅行の宿泊数や目的地（国内・海外）に合わせて持ち物リストを自動生成し、URLひとつで仲間と共有・編集できるWebアプリケーションです。

アプリのインストール不要、ブラウザだけで完結するシンプルな設計で、グループ旅行や家族旅行の準備をスムーズにします。

## ✨ 主な機能

* **📋 リスト自動生成**
    * 「国内/海外」と「宿泊数」を選択するだけで、最適な持ち物リストを自動で提案します。
* **✏️ 自由な編集 & 0から作成**
    * アイテムの追加・削除は自由自在。「自分で0から作成する」モードも搭載。
* **🖐️ ドラッグ＆ドロップ並べ替え**
    * アイテムの順番を直感的に入れ替え可能。スマホタッチ操作にも完全対応。
    * 並び順はデータベースに保存され、共有相手にも同じ順番で表示されます。
* **🔗 カンタン共有**
    * 発行されたURLをLINEなどで送るだけ。ログイン不要で誰でも閲覧・編集が可能。
    * 「URLをコピー」「URLを開く」ボタンでスムーズに共有できます。
* **📱 レスポンシブデザイン**
    * PC、タブレット、スマートフォン（iOS/Android）のどの端末でも見やすく操作しやすいUI。
* **💾 リアルタイム保存**
    * チェックボックスの状態やリストの変更は即座にデータベースへ反映されます。

## 🛠 使用技術

### Frontend
* **HTML5 / CSS3**: モバイルファーストなレスポンシブデザイン。
* **Vue.js 3**: CDN版を使用（ビルド不要）。リアクティブなUI操作を実現。
* **SortableJS**: 滑らかなドラッグ＆ドロップ機能の実装。

### Backend
* **PHP 7.4+**: シンプルなREST APIエンドポイントとして機能。
* **MySQL / MariaDB**: リストデータとマスタデータの管理。
