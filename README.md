# 📦 Packa (パッカ)

**~ 旅の準備を爆速にする、共有パッキングリストツール ~**

Packaは、旅行の目的地や宿泊数に合わせて持ち物リストを自動生成し、URL一つで仲間と共有・編集できるWebアプリケーションです。

## ✨ 主な機能
- **自動リスト生成**: 国内/海外の切り替え、および宿泊数に応じた衣類等の数量自動計算。
- **海外旅行最適化**: パスポートや変換プラグなど、海外旅行特有の必須アイテムを強調表示。
- **共有・共同編集**: 発行されたURLをシェアするだけで、複数人でチェック状態をリアルタイム（保存時）に同期。
- **直感的なUI**: チェック時の打ち消し線、アイテムの追加・削除機能など、ストレスのない操作感。
- **セキュリティ**: HTTPS強制リダイレクト、DB接続情報の隠蔽、XSS/SQLインジェクション対策。

## 🛠 技術スタック
- **Frontend**: Vue.js 3 (Composition API)
- **Backend**: PHP 8.x
- **Database**: MySQL 8.0
- **Styling**: CSS3 (Responsive Design)
- **Deployment**: ロリポップ！レンタルサーバー

## 🚀 セットアップ / 導入方法
1. MySQLで `items_master`, `shared_lists`, `list_items` テーブルを作成（SQLファイル参照）。
2. `db_config.php` にデータベース接続情報を記述。
3. サーバーにアップロードし、HTTPSを有効化。

## 🔒 セキュリティについて
- `.htaccess` による設定ファイルへの直接アクセス禁止。
- サーバー環境（Local/Production）に応じたDB接続情報の自動切り替え。
- PHPによる強制HTTPSリダイレクトの実装。
